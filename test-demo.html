What We Built:

MEV Shield Agent - A Cloudflare Worker that protects crypto transactions from front-running by routing them through private relays instead of the public mempool.

Components:

  • Landing Page - Beautiful website explaining the service
  • RPC Endpoint - Where wallets connect to get protection
  • Status Tracking - Monitor transaction status

Test the Agent:

How It Works:

1. User sends transaction through wallet
2. MEV Shield receives the transaction
3. Routes through private relays (Flashbots, bloXroute, Eden)
4. Bypasses public mempool (no front-running)
5. Returns transaction hash to user
6. Tracks status for monitoring
